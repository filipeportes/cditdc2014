Injeção no CDI


Por exemplo, ao criarmos uma classe simples com um método que retorna uma String: 

                public class Greeting {
                    public String greet(String name) {
                        return "Ola, caro " + name + ".";
                    }
                }
Essa classe já pode ser injetada em qualquer outra utilizando a anotação @Inject.
Vamos injeta-la no managedBean desta página JSF!! 
                @Named
                @ViewScoped
                public class InjecaoBean {
                    @Inject
                    private Greeting greeting;

                    ...
                }
