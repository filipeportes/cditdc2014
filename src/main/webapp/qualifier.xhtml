CDI Qualifiers


Você pode usar qualifiers para prover mais de uma implementação pra um tipo específico de Bean, um Qualifier é basicamente uma anotação Java.

Por exemplo podemos declarar um Qualifier chamado @Informal: 

                import javax.inject.Qualifier;

                @Qualifier
                @Retention(RetentionPolicy.RUNTIME)
                @Target({ElementType.TYPE, ElementType.METHOD, ElementType.FIELD, ElementType.PARAMETER})
                public @interface Informal {}
em seguida aplicá-lo a uma classe que extenda a classe Greeting: 
                @Informal
                public class InformalGreeting extends Greeting {
                    public String greet(String name) {
                        return "E ai " + name + ", Beleza!!.";
                    }
                }
