Escopos

O Escopo de um Bean define qual será seu ciclo de vida, mais especificamente, quanto uma instância dele deve "sobreviver" para que possa ser reutilizada,
a especificação do CDI define os sequintes escopos:

Escopo Duração

@RequestScoped

Uma interação do usuário com uma página web em um único HTTP request

@SessionScoped

Uma interação do usuário com uma página web através do tempo de sessão dessa.

@ApplicationScoped

Estado compartilhado entre todos os usuários interagindo com uma aplicação web.

@Dependent

É o escopo padrão se nenhum outro for definido,um objeto existe para servir um cliente(bean)
e tem o mesmo ciclo de vida dele(bean).

@ConversationScoped

O escopo conversation existe entre limites definidos pelo desenvolvedor,
que podem se extender através de diversos requests,
porém não podem ultrapassar o limíte da sessão HTTP.
