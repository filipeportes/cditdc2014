Injeção no CDI


CDI torna possível injetar uma variedade de objetos, e injetar até mesmo em objetos que não são controlados pelo container Java EE

Os seguintes tipos de objetos podem ser injetados:

  • (Praticamente) Qualquer classe Java
  • Session Beans
  • Java EE resources: data sources, Java Message Service topics, queues, connection factories, e outros
  • Persistence contexts (EntityManager)
  • objetos retornados por métodos anotados com @Producer
  • referências a Web service
  • referências a Remote enterprise beans
